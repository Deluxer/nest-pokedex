# Run in Dev
1. Clone repo
```
git clone git@github.com:Deluxer/nest-pokedex.git
```

2. install
```
yarn install
```
3. Install Nest CLI
```
npm i -g @nest/cli
```

4. Create Database with docker
```
docoker-compose up -d
```

5. Seed population Database
```
{{url}}/api/v2/seed
```

## Stack used
* MongoDB
* Nest
